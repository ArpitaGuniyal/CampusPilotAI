# CampusPilotAI

An AI-powered smart college assistant built using LangChain, Gemini, RAG, FAISS, and Streamlit.

## Features

- Student Assistant
- Faculty Assistant
- College Information Retrieval
- RAG-based Question Answering
- Timetable Assistance
- Attendance Queries
- Library Queries

## Tech Stack

- Python
- Streamlit
- LangChain
- Google Gemini
- FAISS
- Pandas
- NumPy
